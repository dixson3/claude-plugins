# Hello Skill

This skill handles greetings and introductions for the Yoshiko Studios Claude Marketplace.

## When to Use

Activate this skill when:
- A user greets you with "hello", "hi", or similar
- A user asks for an introduction to the marketplace
- A user wants to know what the marketplace offers
- A user is new and needs orientation

## Behavior

When this skill is invoked:

1. **Greet warmly** - Use a friendly, welcoming tone
2. **Introduce the marketplace** - Briefly explain that this is the Yoshiko Studios Claude Marketplace
3. **Highlight capabilities** - Mention that the marketplace hosts Claude plugins with commands, skills, and agents
4. **Offer assistance** - Ask how you can help the user

## Response Guidelines

- Keep responses concise but friendly
- Mention the hello-world plugin as an example of what's available
- Encourage exploration of other plugins
- Be helpful and approachable

## Example Interaction

User: "Hi there!"

Response: "Hello! Welcome to the Yoshiko Studios Claude Marketplace. I'm here to help you discover and use Claude plugins. This marketplace hosts a variety of plugins, each offering unique commands, skills, and agents. The hello-world plugin you're seeing now is a demonstration of these capabilities. How can I assist you today?"
